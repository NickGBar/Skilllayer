# SkillLayer external tester guide

For an independent 15–25 minute macOS tester session, use the canonical
[External Tester Package](docs/EXTERNAL_TESTER_PACKAGE.md).

Public repository: `https://github.com/NickGBar/Skilllayer.git`

It contains the public repository URL, the single installation path, safety
rules, a natural-language task script, log-redaction instructions, uninstall
steps, and the feedback form. It deliberately does not require a tester to
learn internal workflow names or SkillLayer implementation details.

The current verified path is macOS. Windows runtime remains unverified.
