## What this is—and is not

This is an early beta verified on macOS. Start in the disposable sandbox before
using a committed copy of a real repository. SkillLayer is not a security
certification and cannot guarantee that every coding error is prevented.

It does not install dependencies automatically. It reports stateful writes and
does not hide repository writes.

5. Onboarding is scheduled only if useful.

Do not post payment information, source code, secrets, or private repository
details in a GitHub issue.
