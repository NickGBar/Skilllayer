# SkillLayer 0.2.0 release notes

SkillLayer 0.2.0 prepares the early-access release foundation: professional skills, one-prompt installation, truthful environment handling, consistent version reporting, and local sanitized diagnostics.

It does not add automatic dependency installation, telemetry upload, or a guarantee of safety. Start in the disposable sandbox and commit or back up real work before first use.

See [CHANGELOG.md](CHANGELOG.md) and [DIAGNOSTICS.md](DIAGNOSTICS.md).
