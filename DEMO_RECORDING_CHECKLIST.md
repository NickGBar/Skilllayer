# Demo recording checklist

- [ ] Start with a clean, committed sandbox.
- [ ] Hide usernames, absolute paths, tokens, keys, and private tabs.
- [ ] Use no production or private repository.
- [ ] Use a readable terminal font and zoom level.
- [ ] Show the actual prompts and key structured outputs.
- [ ] Cut dependency-install and test waiting time; do not fake output.
- [ ] Keep the final video between 60 and 120 seconds.
- [ ] Add captions.
- [ ] Review video metadata before publishing.
- [ ] End with the one-prompt installation CTA and free early-access note.
