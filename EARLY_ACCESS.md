# SkillLayer free early access

SkillLayer is currently available as **free early access**. No payment or
account is required.

The early beta includes the current professional skills:

- Safe Code Change
- Release Readiness
- Resume Project Work

## Start safely

macOS is currently verified. Begin with the disposable
[tester sandbox](https://github.com/NickGBar/skilllayer-tester-sandbox) if you
want to evaluate SkillLayer before using a committed copy of a real repository.
Stateful writes are disclosed, SkillLayer does not edit `.gitignore`, and it
never installs dependencies automatically.

## Optional feedback

Feedback is optional. If you choose to share it, review and redact it first;
SkillLayer never uploads results automatically. Use the public
[early-access feedback form](https://github.com/NickGBar/Skilllayer/issues/new?template=beta-interest.yml)
and do not include source code, credentials, `.env` contents, private paths,
or unrelated MCP configuration.
